# Jellyfin Docker Container

## Content
The repository contains the following files
```
├── docker-compose.yaml
└── README.md
```

## Compose yaml
The docker compose yaml file is based on the example found in the official documentation:
https://jellyfin.org/docs/general/installation/container/#installation-instructions

## Control container
The container can be started with
```
docker compose up [-d]
```
and stopped with
```
docker compose stop jellyfin
```
